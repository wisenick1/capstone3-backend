# capstone3-backend
2024학년도 2학기 캡스톤3 프로젝트 백엔드

## 🧰 변경 타입 - PR template
* [o] 🤹 FEATURE	: 기능 추가  
* [ ] 🔧 FIX		: 버그 수정  
* [o] 🗑️ DELETE	: 파일 삭제  
* [ ] 🎨 DESIGN	: 디자인 작업  
* [ ] 💬 COMMENT	: 주석 작성  
* [ ] 📄 DOCS 	: 문서 작업  

## ✒️ 변경 내용
* Groovy기반의 파일을 Java기반으로 수정
* 기존의 .gitignore 은 수정이 반영된 버전
* application.properties의 숨겨놓은 항목을 수정 후 테스트 요망

## 📢 주의 및 리뷰 요청
* localhost:8080 으로 테스트, 주말 즈음에 NaverMap 추가될 예정
